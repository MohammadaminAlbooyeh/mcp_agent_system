# Tools Guide

## Web Tools
- `web_search` - Search the web
- `web_scraper` - Extract content from pages
- `url_fetcher` - Fetch URL content

## Database Tools
- `sql_executor` - Execute SQL queries
- `db_reader` - Read from database
- `db_writer` - Write to database

## File Tools
- `file_reader` - Read files (PDF, TXT, CSV, DOCX)
- `file_writer` - Write files
- `file_manager` - Copy, move, delete files

## Email Tools
- `gmail_reader` - Read Gmail emails
- `gmail_sender` - Send emails
- `email_parser` - Parse email content

## API Tools
- `http_client` - Make HTTP requests
- `rest_caller` - Call REST APIs
- `webhook_handler` - Send webhooks

## Code Tools
- `code_executor` - Execute Python code
- `code_analyzer` - Analyze code
- `test_runner` - Run tests

## Utility Tools
- `calculator` - Math calculations
- `datetime_tool` - Date/time operations
- `text_processor` - Text transformation
- `data_formatter` - Data format conversion
