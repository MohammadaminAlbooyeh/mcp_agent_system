Core Components

🤖

Agent Core

Central orchestration engine managing the execution loop and tool coordination.

  • Task parsing & planning
  • Tool orchestration
  • Error handling
  • State management
🧠

Reasoning Engine

Implements ReAct pattern with chain-of-thought reasoning for multi-step problem solving.

  • ReAct loop
  • Chain-of-thought
  • Self-reflection
  • Adaptive strategy
💾

Memory System

Dual-layer memory for context awareness and long-term knowledge persistence.

  • Short-term context
  • Long-term knowledge
  • Semantic search
  • Memory consolidation
🔌

MCP Server

Model Context Protocol server providing access to 15+ specialized tools.

  • Tool registration
  • Protocol handling
  • Resource management
  • Error recovery
⚡

REST API

FastAPI-based REST interface for external system integration and control.

  • Task submission
  • Status queries
  • Result retrieval
  • WebSocket support
🎨

React Frontend

Real-time monitoring dashboard for agent execution and system health visualization.

  • Live execution view
  • Tool monitoring
  • Performance metrics
  • Agent control
